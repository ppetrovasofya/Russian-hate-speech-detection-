Выпускная квалификационная работа по автоматическому распознаванию языка вражды в русскоязычных комментариях.

---

## 📌 Кратко

Сравниваются два подхода:

- **Классический ML**: TF‑IDF + логистическая регрессия (baseline)
- **Современный NLP**: дообученный **RuBERT** (DeepPavlov/rubert-base-cased)

Иерархическая двухуровневая классификация:

1. **Бинарный**: *hate* vs *no‑hate*
2. **Мультиклассовый**: *ethnic*, *religious*, *sexism*, *ableism*

---

## 📊 Данные

- **Источники**: открытые датасеты + ВКонтакте + Telegram (новостные паблики)
- **Объём**: 8 694 размеченных текста
- **Согласие аннотаторов**: Cohen’s κ = 0.71 (ур. 1) и 0.83 (ур. 2)

| Категория | Количество | Доля |
|-----------|------------|------|
| Hate speech | 3 916 | 45% |
| No hate speech | 4 778 | 55% |

---

## 🧹 Предобработка

- **Baseline**: очистка → нижний регистр → токенизация (NLTK) → лемматизация (pymorphy3) → удаление стоп-слов → TF‑IDF (униграммы + биграммы)
- **RuBERT**: только базовая очистка; токенизация и стоп-слова не удаляются (механизм внимания)

---

## 🤖 Модели

### Базовый классификатор
- Logistic Regression, `C=0.05`, `class_weight='balanced'`

### RuBERT
- `DeepPavlov/rubert-base-cased` (~110 млн параметров)
- Fine‑tuning: 5 эпох, lr=2e‑5, batch=8, max_length=128
